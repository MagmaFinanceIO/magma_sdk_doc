# Retrieve almm pools

## 1. Retrieve all pools

Method: `magmaClmmSDK.Almm.getPools()`

### Example

```typescript
async function retrievelAllPools() {
  const almmPools = await mainnetSDK.Almm.getPools();
  console.log(`pool length: ${almmPools.length}`);
}

// retrievelAllPools()
// pool length: 82
```
